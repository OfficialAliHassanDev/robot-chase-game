# 🤖 Robot Chase Game

A JavaScript mini game where the player controls a human character while escaping from an AI robot.

## Features

- Arrow Key Controls
- AI Robot Chasing System
- Collision Detection
- Boundary Detection
- Start Button
- Responsive Design

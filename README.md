# 🤖 Robot Chase Game

A fun JavaScript-based mini game where the player controls a human character using arrow keys while avoiding an AI-powered robot that continuously chases the player.

## 🎮 Features

* Human movement using Arrow Keys
* AI Robot Chasing System
* Collision Detection
* Boundary Detection
* Game Over Screen
* Start Game Button
* Responsive Design
* Interactive Gameplay

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Play

1. Open the game in your browser.
2. Click the **Start Game** button.
3. Use the **Arrow Keys** to move the human character.
4. Avoid touching the game boundaries.
5. Avoid getting caught by the robot.
6. If the robot catches you or you hit a boundary, the game is over.

# 🤖 Robot Chase Game

A fun JavaScript survival game where the player controls a human character and tries to escape from a chasing robot while collecting coins and leveling up.

---

## 🎮 Features

✅ Keyboard Controls (Arrow Keys)

✅ Mobile Swipe Controls

✅ Coin Collection System

✅ Score System

✅ Level System

✅ Level Up Popup

✅ Robot AI Chasing System

✅ Collision Detection

✅ Boundary Detection

✅ Mario Coin Sound Effect

✅ Mario Death Sound Effect

✅ Responsive Design

## 🎯 Objective

- Collect coins to increase your score.
- Avoid the robot.
- Stay inside the game area.
- Reach higher levels by collecting more coins.

---

## 🕹️ Controls

### Desktop

| Key | Action |
|------|---------|
| ⬅️ | Move Left |
| ➡️ | Move Right |
| ⬆️ | Move Up |
| ⬇️ | Move Down |

### Mobile

| Swipe | Action |
|--------|---------|
| 👉 Swipe Right | Move Right |
| 👈 Swipe Left | Move Left |
| 👆 Swipe Up | Move Up |
| 👇 Swipe Down | Move Down |
